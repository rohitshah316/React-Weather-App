# 🌤️ React Weather App

A clean and responsive **React weather application** that allows users to search for any city and view real-time weather information.  
The app fetches live weather data from the **OpenWeather API** and displays it with a modern and minimal UI.

Users can quickly **search for a city and see current weather conditions** including temperature and other important details.

## 🛠️ Features

- Search weather by city name  
- View real-time temperature and weather conditions  
- Displays weather details like humidity, wind, and condition  
- Fully **mobile responsive design**  
- Fast performance using **Vite**  
- Clean and modern UI with **Tailwind CSS**

---

## ⚙️ Tech Stack

- **React JS**
- **Vite**
- **Tailwind CSS**
- **OpenWeather API**
- **JavaScript (ES6+)**
- **Responsive Design**
